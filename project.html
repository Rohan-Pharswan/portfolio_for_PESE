An E-Attendence System that uses Blockchain

An electronic attendence system that uses blockchain to ensure tamper proof attendence information of students, professors and employees. It uses concepts of blockchain to securely store data into blocks that can't be changed externally, which makes it tamper proof, available for all and eliminate the need to make a seperate registery which can be easily changed by others
